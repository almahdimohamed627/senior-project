** Deployment & DevOps
1.How many Docker containers are deployed, and what are their responsibilities?
- 6.
- traefik: reverse proxy and load balancer.
- db: main and only database.
_ ai-model: diagnosis for teeth camara pictures for 3 diagnosis.
- ai-agent: talks to patient to diagnosis him to the right lab of dental study.
_ fusionauth: handles authentication, jwt, account verification, role management.
- backend: core routing system and handle request/session, chat and stuff.
  
2. What happens when a CI/CD pipeline stage fails?
- if a component fails the pipeline dont fail.
_ there are tree stages: build, deploy and integration testing.
- there are a nice report that sends automaticlly to telegram.
_ if a stage failed to reports failier.
- if build failed for any component it skips deployment for this component.
- every success is reported, every failier is reported.
  
3. How is system health monitored after deployment?
- every component build have health check
_ every component deploy have responding check.
- and there is integration testing for all component integration.
